# Cloud Governance Platform

An AI-powered cloud governance platform for compliance, security, cost optimization, and resource management.

## Features

- Compliance validation
- Cost optimization
- Security governance
- Resource governance
- Governance workflow orchestration
- FastAPI backend
- Docker support

- Cost Agent

- Governance Workflow
- AWS Scanner
- Policy Engine
- Cost Analyzer
